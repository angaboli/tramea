Principes

Ce qui guide chaque écran

Tramea est un outil de régie utilisé en conditions réelles, pendant le culte. Le design transmet d'abord la confiance.

Fiabilité

Un outil de régie ne flanche jamais en plein culte. État du système toujours visible, sauvegarde automatique.

Simplicité

Chaque écran va à l'essentiel. Pas d'effet décoratif inutile, pas de surcharge : l'information utile, lisible.

Rapidité d'usage

Construire une trame en quelques clics. Les actions fréquentes sont grandes, accessibles, sans détour.

Clarté opérationnelle

L'état de chaque élément — brouillon, validé, exporté — se lit d'un coup d'œil grâce à un langage de statuts cohérent.

Hors-ligne d'abord

Tout fonctionne sans connexion. La synchronisation attend son tour ; l'utilisateur n'est jamais bloqué.

Chaleur maîtrisée

Sérieux et structure dans le bleu, énergie et chaleur dans l'orange — réservé aux actions et accents qui comptent.

12 — Accessibilité

Lisible par tous, en toute condition

En régie, la lumière est faible et le temps est compté. Le contraste et la taille des cibles ne sont pas négociables.

AA+
Contraste

Texte courant ≥ 4,5:1, gros titres ≥ 3:1. Le bleu 600 sur blanc atteint 7,4:1.

44px
Cibles tactiles

Toute zone cliquable mesure au moins 44 × 44 px, espacée de 8 px.

3px
Focus visible

Anneau de focus net sur tout élément interactif, navigable au clavier.

≠ ●
Jamais la couleur seule

Chaque statut combine couleur, pastille et libellé texte.

13 — Tokens techniques

Prêts pour Vite + React + Tailwind

Les tokens vivent en variables CSS, consommées par Tailwind. Le thème sombre s'active via data-theme="dark" ; aucune valeur en dur dans les composants.

tokens.css
/* Couleurs sémantiques — thème clair */
:root {
  --color-primary: #2F557F;
  --color-primary-hover: #284769;
  --color-accent: #FF6C1A;
  --color-bg: #F6F8FA;
  --color-surface: #FFFFFF;
  --color-border: #E2E6EB;
  --color-text: #1A2129;
  --color-success: #1F8A5B;
  --color-warning: #C9881A;
  --color-error: #D64545;
  --radius: 10px;
}
/* Thème sombre */
[data-theme="dark"] {
  --color-bg: #0E1116;
  --color-surface: #171C23;
  --color-text: #EAEEF3;
  --color-primary: #4A77A6;
}
tailwind.config.ts
export default {
  darkMode: ["selector", '[data-theme="dark"]'],
  theme: {
    extend: {
      colors: {
        primary: "var(--color-primary)",
        accent: "var(--color-accent)",
        bg: "var(--color-bg)",
        surface: "var(--color-surface)",
        border: "var(--color-border)",
      },
      borderRadius: {
        DEFAULT: "var(--radius)",
      },
      fontFamily: {
        sans: ["Hanken Grotesk", "sans-serif"],
        mono: ["JetBrains Mono"],
      },
    },
  },
}

Store Zustand recommandé pour l'état de la trame et la file de synchronisation hors-ligne ; les composants ne lisent que des tokens, jamais de couleurs en dur — un seul fichier tokens.css pilote toute l'identité.
